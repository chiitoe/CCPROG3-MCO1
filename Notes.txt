RESTRICTIONS AND RULES
	- no libraries outside of Java21 API
	
	AFFILIATION AND CHARACTER:
	- faction exclusivity - character cannot be a pirate and a marine simultaneously
	- single group allegiance - pirate can only belong to ONE pirate crew and marine can only belong to ONE marine corps at a time
	- command validation - pirate crew can only have ONE captain. newly assigned captains will automatically revert the old captain's status back to the standard role
	
	DEVIL FRUIT:
	- a devil fruit can only have ONE current owner at a time. current owner can only reliquish the devil fruit if they are marked as "dead"
	- when a character is pronounced "dead", the system must IMMEDIATELY execute:
		- append dead character to fruit's historical owners list
		- set current owner to null
		- clear devil fruit power field on the dead character's profile
